IHateWork

Cross-platform local media processing tool. Process PDFs, videos, audio, and images completely offline. Free and open-source.

Download for macOS Download for Windows Download for Linux

Features

PDF

PDF Tools

Merge, split, compress, rotate, add watermarks, encrypt, and decrypt PDF files.

Video

Video Processing

Compress videos to reduce file size. Convert between popular video formats.

Audio

Audio Conversion

Convert audio files between formats. Extract audio tracks from video files.

Image

Image Tools

Create GIFs, resize, crop, rotate, flip images. Extract colors from images.

Installation

macOS (Homebrew)

brew tap maplex18/ihatework
brew install --cask ihatework

Windows (Winget)

winget install maplex18.IHateWork

Linux (Snap)

snap install ihatework
